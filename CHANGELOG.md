# Changelog

All notable updates to this guide are documented in this file.

## 2026-W13

- Added core tools & platforms (InnoClaw, ResearchClaw, FARS, Orchestra, OpenClaw, EvoScientist).
- Added Deep Research tools (OpenAI/Gemini/Perplexity), OpenAI Prism, MCP Servers for research.
- Switched all content to English as default language.
- Expanded to 35+ papers across 9 topic files.

## 2026-W12

- Added a new `topics/claw-park.md` guide to explain what each Claw project is building and how the ecosystem layers fit together.
- Added a 2026 landscape snapshot to `README.md` to clarify the current split between research copilots, autonomous AI scientists, and vibe coding execution layers.
- Added a new `topics/vibe-coding.md` guide covering terminal agents, coding agents, background agents, and repo safety patterns.
- Added a new `topics/vibe-anything.md` guide for adjacent prompt-native workflows (apps, design, writing, slides, and ops), explicitly separated from core Vibe Research.
- Expanded `topics/tools.md` with Prism, Claw Park, Vibe Coding tools, and adjacent agentic workflow references.

## 2026-W12 (Reorganization)

- Restructured project to hub-and-spoke architecture inspired by [Embodied-AI-Guide](https://github.com/TianxingChen/Embodied-AI-Guide).
- README.md rewritten as lightweight index hub with numbered bilingual sections.
- Created `topics/` directory with 5 dedicated topic files:
  - `surveys.md`, `systems.md`, `ideation.md`, `synthesis.md`, `benchmarks.md`
- Each topic file includes full paper metadata, reading tips, and extended reading resources.
- Added new sections: News, Useful Info, Citation, Star History.
- Added `files/images/` directory for future static assets (logo, etc.).
- Removed redundant `Vibe Research 入门阅读清单.markdown` (content merged into topics/).
- Supplemented Useful Info with courses, communities, awesome lists, and conferences.

## 2026-W12 (Initial Release)

- Public launch of the repository.
- Added bilingual `README.md` as the canonical entry point.
- Added structured reading path and categorized paper tables.
- Added contribution workflow:
  - `docs/curation-guidelines.md`
  - `.github/ISSUE_TEMPLATE/resource_suggestion.yml`
  - `.github/PULL_REQUEST_TEMPLATE.md`
