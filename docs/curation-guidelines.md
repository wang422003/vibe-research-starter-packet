# Curation Guidelines

This document defines how resources are selected and maintained in the Vibe Research Starter Packet.

## Scope

We curate resources related to:

- LLM agents for scientific discovery
- AI-assisted research ideation and planning
- Literature synthesis for research workflows
- Scientific benchmarks and evaluation for research agents

## Inclusion Criteria

A resource should satisfy most of the following:

1. **Relevance**: Clearly connected to scientific discovery workflows or research agents.
2. **Representativeness**: Introduces an important method, framework, benchmark, or perspective.
3. **Evidence Quality**: Includes reasonably rigorous evaluation or clear empirical support.
4. **Practical Learning Value**: Helps newcomers build reusable understanding.

## Suggested Metadata (Required in PRs)

- Title
- URL (paper/project/blog/repo)
- Type (Survey, System, Ideation, Benchmark, etc.)
- Why read (1-2 sentences)
- Suggested stage (Beginner / Beginner-Intermediate / Intermediate)
- Keywords (2-5 tags)

## Exclusion Signals

Usually not included:

- Off-topic resources with weak connection to Vibe Research
- Marketing-only content without technical depth
- Duplicates without meaningful additional value
- Claims that cannot be reasonably verified

## Update Policy

- Keep the list concise and high-signal
- Prefer quality over volume
- Periodically re-evaluate old entries
- If a better resource supersedes an older one, update with rationale in PR

## Contribution Process

1. Open an issue using the `Resource Suggestion` template.
2. Discuss fit and placement category.
3. Submit a PR using the PR template and fill all metadata.
4. Maintainers review for scope, quality, and duplication.
